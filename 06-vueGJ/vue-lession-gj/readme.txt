库 和 框架区别？
库：工具箱，一般是调用库中的api
框架：毛坯房，我们在指定的位置写好代码，框架去调用  框架是库的升级版  

声明式 和 命令式？
命令式：for 循环  for(let i = 0; i<=10; i++){}
声明式：reducer  map  some  every   map(item=>{})  v-if
