APP常见的几大模块：
    1，首页面
        轮播图
        商品列表《垂直列表，水平列表》
        搜索框（假的）---> 搜索组件（真实的框）
                            历史记录（CRUD） 
                            推荐商品（跳转）

    2，商品分类 
        左边的大分类
            基本包含app中所有商品的分类
        右边的小分类
            点击时，一般会跳到某一类的商品列表
        左右联动
        商品搜索

    3，具体某一类商品的列表（台式机）
    
    4，商品详情
        加入购物车，购买

    4，购物车组件
        结算

    5，结算组件 
        地址
    
    6，地址组件 
        CRUD 
    
    7，返回到结算组件
        支付宝，微信

    8，订单组件
        显示购买的东西

    9，我的模块
        用户管理（用户登录，用户注册）
        短信登录，找回密码... 
        查找购买过的商品
        ....

        电商APP
